# iOS-Webp-Code

个人主页：<http://cccsee.cn>

***
####功能
***
实现，iOS项目中，本地加载webp(智图)，支持从SDWebImage，url在线加载webp。

####说明
***
本Demo，没有使用 WebP.framework 和 libwebp.a

而是使用的 完整的源码c代码，对于不喜欢更新lib的朋友比较适用

项目中同时包含了 SDWebImage的源码

使用很简单，就是导入项目就可以了。

在 UIImage+TLWebP.h 文件中，有相关说明
